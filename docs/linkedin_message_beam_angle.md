## How to Use eulumdat.icu for Beam Angle Visualization

### Step-by-step for the author:

1. **Visit**: https://eulumdat.icu

2. **Load a Wikipedia demo template** (dropdown menu):
   - `wiki-batwing.ldt` - **Best for IES vs CIE comparison** (center < max)
   - `wiki-spotlight.ldt` - Narrow 30° beam (standard center-peak)
   - `wiki-flood.ldt` - Wide 60° beam / 120° field

3. **View beam angles**:
   - Look at the polar diagram - batwing shows visible "dip" at center
   - Check beam/field angle values in the info panel
   - For batwing: IES angle (green) vs CIE angle (blue) shown separately

4. **Interactive diagrams**:
   - Polar diagram shows intensity distribution with beam/field overlays
   - Cone diagram shows beam spread at floor level
   - All diagrams exportable as SVG for Wikipedia

5. **Try editing values**:
   - Modify intensity values in the data table
   - Watch beam/field angles update in real-time
   - Demonstrates how distribution shape affects angle calculations
